HTML Linguagem de Marcação de Hipertexto

Tutorial para Iniciantes

HTML (Linguagem de marcação de Hipertexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web . Outras tecnologias do HTML geralmente são usadas para descrever a aparência/apresentaçção (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página web.
